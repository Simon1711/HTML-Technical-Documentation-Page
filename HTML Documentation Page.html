Introduction

HTML stands for Hypertext Markup Language. It is the standard language for creating web pages and web applications.

HTML elements are the building blocks of HTML pages. They define the structure and content of the document.

HTML consists of a series of elements, each of which has a specific meaning and purpose.

Elements are represented by tags, which can contain attributes to provide additional information about the element.

HTML documents are structured hierarchically, with nested elements creating a tree-like structure.

Here is an example of a basic HTML document:

<!DOCTYPE html> <html> <head><title>Document</title></head> <body><h1>Hello World</h1></body> </html>
HTML Elements

HTML elements are the core components of an HTML document. They include headings, paragraphs, links, and more.

Each element is enclosed in angle brackets, such as <div> or <a>.

Some elements are empty, meaning they donâ€™t have closing tags. Examples include <br> and <img>.

Elements can be nested inside other elements to create more complex structures.

HTML5 introduced new semantic elements like <article> and <section> to enhance the structure and readability of documents.

  • <header> - Represents introductory content or a set of navigational links.
  • <footer> - Represents the footer of a section or page.
  • <main> - Represents the dominant content of the <body>
Attributes

Attributes provide additional information about HTML elements. They are included in the opening tag of an element.

Attributes are written as name-value pairs, such as href="https://example.com" for a link element.

Some attributes are global, meaning they can be used with any HTML element. Examples include class and id

Other attributes are specific to certain elements. For example, the src attribute is used with the <img> element to specify the image source.

Here is an example of an img element with attributes:

<img src="image.jpg" alt="Description">
Forms

Forms are used to collect user input. They consist of various input elements like text fields, checkboxes, and submit buttons.

The <form> element is the container for all form controls.

Forms can use methods like GET or POST to send data to a server.

Common form elements include <input>, <select>, and <textarea>

Forms often include validation to ensure that the data entered is correct before submission.

Here is a basic form example:

<form action="/submit" method="post"> <input type="text" name="username"> <input type="submit" value="Submit"> </form>
Tables

Tables are used to organize data into rows and columns. They consist of the <table> element.

Each row in a table is represented by the <tr> element.

Table headers are defined with the <th> element, while table data is defined with the <td> element.

Tables can be styled with CSS to enhance their appearance and make them more readable.

  • <tr> - Represents a table row.
  • <th> - Represents a table header cell.
  • <td> - Represents a table data cell.
